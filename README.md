## Install

1. Clone the project.

```sh
git clone https://petronasvsts.visualstudio.com/PDB%20AWSM/_git/PDB-AWSM-Frontend
```

> **NOTE**: Or you use the CLONE IN VSCODE features provided.

<br/>

1. Run the npm command below to install the project dependecies.

```sh
npm install
```
